In this video https://youtu.be/m2ddi9shfwA, I modified the Hugging Face Tiny Whisper script called RunWhisper.cs to accept input from your microphone. Before adding this to your project, please go here https://huggingface.co/unity/inference-engine-whisper-tiny and set it up according to the instructions on the site and the walkthrough in my video.

Once you successfully transcribed the provided 16KHz .wav file, then delete all the contents of the RunWhisper.cs file and copy and paste the RunWhisper.cs file in the repositiory.
